Restful Pokedex Homework

Written by Wei

Went through homework, and went through my bug. Not entirely sure how it works, but now I know where to look to see exactly what happened. I actually sorta wanna know how the back end of Ruby works, because it doesn’t quite make sense to me. Though tbh functionally it isn’t a problem- I can just work around it.

So today we did Sinatra. Feels like we did all of what we did in express in Sinatra in a day. Pretty heavy. Getting data from forms, make controllers etc etc etc. I got really distracted halfway through since it really feels just like doing the same thing again.

For homework, we were supposed to make a randomiser. I’d done that before in a lot of my previous homeworks, and specifically a compliment generator I’d made in like week one. So instead, Alex was like… make a pokedex. So I made all the functionality of a pokedex, but really looking back on it I should have made the design of one instead. Oh well.

Not much to say. Homework wasn’t groundbreaking, but I spent such a long time building the infrastructure, and in the end didn’t do enough CSS, I feel. I probably should spend a lot more time on CSS. I did, however, learn how to make circles and stuff… so there is that. I still need to learn how to do responsive design.
