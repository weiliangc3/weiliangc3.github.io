Pig Latin code test Friday

Written by Wei

So Fridays are slow, especially if there’s nothing I really want to ask or find out about, and I don’t really have that many questions this week. Instead we did a code test, like exercism or something? We did a pig latin translator which is quite fun. Approaching logical problems is always quite fun, and you always learn something- if not about approach then about potential syntax that you can use to make code either dryer or more readable. Quite good.

Then we did object oriented blackjack, to get used to object oriented stuff, and we ended the day with a grand national bet randomiser for the class.
